# 📊 AI Sales Forecast System

An AI-powered **Sales Prediction and Business Analytics System** built using **Machine Learning, Streamlit, MySQL, and Firebase**.

## 🚀 Features

* User Authentication (Login / Register / Google OAuth)
* AI-based Sales Prediction
* Interactive Dashboard using Streamlit
* MySQL Database Integration
* Firebase Realtime Database + Firestore
* Machine Learning Model for Forecasting

## 🛠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* MySQL
* Firebase
* NumPy

## 📂 Project Structure

```
sale_cast/
│
├── app.py
├── requirements.txt
├── sales.xlsx
├── database/
├── firebase/
├── model/
```

## ▶ Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
